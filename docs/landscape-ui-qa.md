# 가로모드 UI 디자인 QA

작성일: 2026-05-12

## 기준 화면

- 레노버 샤오신패드 2024 12.7인치 대응 비율 기준
- 가로모드 16:10 화면
- 스크롤 배경: `iv_scroll.png`

## 확인 결과

- 스크롤 배경의 위치와 크기는 유지한다.
- 기존 UI 콘텐츠가 스크롤 이미지의 좌측 찢어진 가장자리 영역까지 침범해 보였다.
- 상단 탭, 공지 티켓, 리스트 아이템을 스크롤 안쪽 종이 영역으로 들여야 했다.
- 리스트의 기본 스크롤바가 스크롤 이미지 위에서 흰색 라인처럼 보여 몰입감을 해쳤다.
- 이미지가 없는 가이드성 항목에 빈 이미지 아이콘이 반복되어 리스트가 지저분해 보였다.
- 타로 시그니처 칵테일 카드가 하단 말림 영역에 가까워 카드 선택창의 여백이 답답했다.

## 반영 사항

- 가로모드 전용 그리드 여백을 추가해 스크롤 안쪽 종이 영역에 탭, 공지, 리스트를 정렬했다.
- 가로모드 리스트 아이템 높이, 배경 농도, 내부 이미지 크기, 가격 글자 크기를 조정했다.
- 리스트 기본 스크롤바를 숨겨 스크롤 이미지와 겹쳐 보이지 않게 했다.
- 이미지가 없는 메뉴 항목은 이미지 영역 자체를 숨기도록 했다.
- 가로모드 타로 카드 크기를 줄이고 화살표/카드 스테이지 폭을 조정했다.

## 남은 체크

- 실제 태블릿 설치 화면에서 좌측 카테고리 버튼과 스크롤 배경 사이의 간격을 최종 확인한다.
- 와인/위스키처럼 실제 이미지가 많은 카테고리에서 리스트 밀도와 가격 가독성을 한 번 더 확인한다.
- 타로 카드 실제 이미지가 연결된 뒤 7:12 비율 카드의 여백을 재확인한다.
