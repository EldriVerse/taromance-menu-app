## 4. 시각 확인 메모

- 포털 타이틀이 1280x720 화면에서 버튼 밖으로 넘치던 문제를 조정했다.
- 타로 카드가 가로 화면에서 하단에 너무 붙던 문제를 줄이기 위해 카드 폭과 stage 높이를 조정했다.
- 현재 UI는 기존 레거시 이미지를 배경으로 사용한다. 다음 단계에서는 기존 이미지를 참고하지 않고 새 16:10 배경 자산을 기획/제작해야 한다.

## 5. 다음 UI 자산 작업 메모

다음 작업은 타이틀 화면 배경과 메인 페이지 배경 이미지를 새로 제작하는 단계다.

기준:

- 태블릿 배경 기준 비율은 16:10
- 가로형 기준 제작
- 세로형은 16:10 원본에서 안전 영역을 고려해 크롭 또는 별도 파생 이미지 검토
- 기존 `iv_mainbg.jpg`, `iv_scroll_1920h_03.png`는 참고하지 않는다.
- 새 컨셉은 매장 분위기, 타로, 매직 스크롤, 게임 접속감을 기준으로 별도 기획한다.

필요 자산:

- 타이틀 화면 배경 16:10
- 메인 페이지 외부 배경 16:10
- 스크롤 패널 내부 배경 16:10 또는 패널 비율 대응 텍스처
- 타로 카드 fallback 7:12
